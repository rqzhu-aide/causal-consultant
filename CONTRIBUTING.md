## What to contribute

- Deeper examples and diagnostics for scaffold subskills
- Domain-specific extensions (biomedical, education, policy, marketing, etc.)
- Validated example datasets and end-to-end reports
- Package-version-specific recipes
- Templated report generation (Quarto, R Markdown, Jupyter)

## How to contribute

1. Fork the repository
2. Create a branch for your changes
3. Make your edits
4. Submit a pull request with a clear description of what changed and why

If you are unsure whether a change fits the design philosophy, open an issue first to discuss.
