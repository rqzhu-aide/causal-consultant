# Result Interpretation Checklist

Before interpreting a result as causal, verify:

- [ ] The estimand is explicitly stated.
- [ ] The target population is stated.
- [ ] The treatment and comparator are well-defined.
- [ ] The outcome scale is scientifically meaningful.
- [ ] Time zero and follow-up are correct.
- [ ] Identification assumptions are stated.
- [ ] Design-specific diagnostics were checked.
- [ ] Sensitivity analyses were considered.
- [ ] The analysis does not adjust for post-treatment variables unless targeting a direct/mediation estimand.
- [ ] The conclusion uses conditional language: “under the stated assumptions.”
- [ ] Limitations are not hidden in appendices only.
