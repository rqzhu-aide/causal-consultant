# User Intake Question Bank

Use only the subset needed for the current project.

## Core Causal Question

1. What treatment, exposure, or intervention do you want to evaluate?
2. What is the comparator or control condition?
3. What outcome do you care about?
4. When should follow-up begin?
5. Over what time window should the outcome be evaluated?
6. What population should the effect apply to?
7. Do you want the average effect, effect among treated, subgroup effects, a treatment rule, a direct/indirect effect, or something else?

## Treatment Assignment

1. Was treatment randomized?
2. If not randomized, how was treatment chosen or assigned?
3. Are there known eligibility criteria or contraindications?
4. Was there noncompliance or crossover?
5. Is there a policy cutoff, instrument, or staggered adoption timing?

## Data Structure

1. What does each row represent?
2. Are there repeated measurements per unit?
3. Is there clustering by site, household, school, provider, state, or network?
4. Are there time-to-event outcomes, censoring, or competing risks?
5. Are there missing data or attrition?
6. Can one unit's treatment affect another unit's outcome?

## Covariates and Timing

1. Which covariates were measured before treatment?
2. Which variables were measured after treatment?
3. Which variables may be mediators?
4. Which variables affect treatment but plausibly not outcome except through treatment?
5. Which variables may affect sample inclusion or follow-up?

## Practical Constraints

1. Do you prefer R or Python?
2. Do you need a full report, code only, or interpretation of existing results?
3. Are there regulatory, clinical, or field-specific reporting standards?
4. Are there variables that cannot be used for ethical, legal, or privacy reasons?
